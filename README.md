geo2fastq
=========

Download FASTQ files from the SRA archive

Currently not in working state :)

## Dependencies

Python:
* Parallel Python
* trackhub
* biopython

To create fastqs:
* sra tools (fastq-dump)
To map:
* soladmin_align.rb
* samtools
To create bigwigs:
* bedtools
* phantompeakqualtools
* R + spp
